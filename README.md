## Getting Started

1. clone this repository
2. do the following steps:
   ``` git
   cd path/to/repo
   cd Assets
   git submodule add https://github.com/HassoPlattnerInstituteHCI/unity-dualpanto-toolkit.git
   git submodule update --init --recursive
   ```
3. test the game in debug mode

> **💡 Tip:**
> When pressing Play, Unity shows a brown screen. Press **"b"** on your keyboard to make the scene visible.
